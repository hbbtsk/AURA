## V1.0.x — RP 引擎核心（当前）

**目标**：一个能跑通的 RP 引擎，角色 50 轮不 OOC，维测可观测

### V1.0.0 必须完成

- [ ] **8 层状态机**
  - [ ] 角色卡导入（SillyTavern PNG 格式）
  - [ ] 8 层 schema 定义 + Pydantic 模型
  - [ ] 每层数值映射到形容词（信任度 0.62 → "基本信任"）
  - [ ] 层间依赖规则（信任度变化触发情绪变化）

- [ ] **双层输出约束**
  - [ ] System Prompt 强制 narrative + structured 格式
  - [ ] structured 层 world_delta 解析 + 执行
  - [ ] 完整性检查（缺 structured 则标记警告）

- [ ] **Director Agent（简化版）**
  - [ ] 意图识别（快模型调用）
  - [ ] 角色调度（谁该说话）
  - [ ] 导演指令生成

- [x] **维测观测台**
  - [x] 实时面板（三元组：人物 + 事件 + 世界）
  - [x] 引擎面板（LLM 调用链路：记忆→压缩→意图→Prompt）
  - [x] 角色切换 + 关系矩阵展示
  - [x] 历史轮次回放（按 chat 多轮次）

- [x] **基础设施**
  - [x] FastAPI 路由层
  - [x] SQLite 表结构（chats、turn_records、角色、事件、世界状态）
  - [x] @snapshot 装饰器（采集中间过程）

### V1.0.1 优化项

- [ ] 声纹 few-shot 注入（角色语言风格锁定）
- [ ] 关系矩阵切换角色联动
- [ ] 质量守卫层（越权检测、风格过滤）
- [ ] ST 卡导入器完善（处理更多字段）

## V1.1.x — 知识图谱（Neo4j）

**目标**：关系矩阵升级关系推理网，因果链升级因果网络

- [ ] **Neo4j 部署**
  - [ ] Docker 本地部署
  - [ ] Python neo4j 驱动集成
  - [ ] 双写策略（SQLite 备份 + Neo4j 查询）

- [ ] **图数据模型**
  - [ ] Character 节点（8层摘要）
  - [ ] Event 节点（含 world_impact）
  - [ ] WorldEntity 节点（场景/物品/组织）
  - [ ] RELATES_TO 关系（有向 + 历史演变）
  - [ ] CAUSED / CONTRIBUTED_TO 关系（强/弱因果）
  - [ ] INVOLVES / AFFECTS 关系

- [ ] **图谱查询 API**
  - [ ] 间接关系查询（A 和 B 通过什么中间节点关联）
  - [ ] 多跳因果回溯（事件的所有间接原因）
  - [ ] 关系演变时序（A→B 的信任度从第 1 轮到第 N 轮）
  - [ ] 全局关系网导出（D3.js 力导向图数据）

- [ ] **维测前端升级**
  - [ ] 关系路径探索面板
  - [ ] 力导向图可视化

---

## V1.2.x — 小说模式

**目标**：从 RP 引擎扩展为通用叙事 AI 工具

- [ ] **大纲 Agent**
  - [ ] 三幕结构自动生成
  - [ ] 章节节拍规划
  - [ ] 作者可手动编辑大纲

- [ ] **叙事 Agent**
  - [ ] 按大纲生成叙述文本（第三人称）
  - [ ] 角色在长篇中保持一致性（50万字+）
  - [ ] 场景描写 + 心理 + 对话混合输出

- [ ] **伏笔追踪**
  - [ ] 自动标记未回收伏笔
  - [ ] 伏笔回收提醒（N 章未提则警告）
  - [ ] 作者手动管理伏笔

- [ ] **作者风格模仿**
  - [ ] 3-5 篇样章提取风格指纹
  - [ ] 句式偏好、修辞习惯、叙事距离
  - [ ] 风格一致性检查

## V1.3.x — 多 Agent 并发

**目标**：物理消除注意力稀释，5+ NPC 同时在线

- [ ] **Director Agent 完整版**
  - [ ] 焦点调度（谁该说话、谁该沉默）
  - [ ] NPC 间信息传递（可见性规则）
  - [ ] 冲突检测与仲裁

- [ ] **多角色 Agent 并行**
  - [ ] 每个 NPC 独立 LLM 调用
  - [ ] Prompt 长度控制（单 Agent < 2.5k tokens）
  - [ ] 输出合并逻辑

- [ ] **RAG 记忆检索**
  - [ ] 语义检索（向量相似度）
  - [ ] 逻辑检索（图谱关联）
  - [ ] 混合排序
